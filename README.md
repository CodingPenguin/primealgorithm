Prime Algorithm
=========

A small terminal app to test whether input is prime or composite number

## Installation

  ```
  $ npm install -g primealgorithm
  ```
  
## Usage

  From the terminal:

  ```
  $ prime -n
  ```
  
  ```
  Number: <Insert Number of Your Choice>
  ```
